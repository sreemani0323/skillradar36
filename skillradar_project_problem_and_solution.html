Problem statement
B.Tech students are applying blind
Every year, thousands of 3rd-year engineering students in India start applying for internships with no clear picture of what the industry actually wants right now. They rely on outdated advice, generic job boards, and guesswork — and then wonder why they aren't getting calls back.
01
No real-time skill visibility
No single place shows which skills are trending across LinkedIn, Internshala, Naukri, and 20+ platforms right now — not last year, not last month, but today.
02
Generic job listings, zero personalization
Existing job boards show you every listing. None of them tell you "here are the 3 skills you're missing to match 80% of AI internships."
03
No trend data — only snapshots
A student can't tell if "LangChain" is a skill worth learning this month or if it peaked 6 months ago. Historical trend data simply doesn't exist for free.
04
No application workflow tool for students
Students track applications in messy spreadsheets or WhatsApp messages. There's no lightweight tracker that lives alongside the job data itself.
Our solution
SkillRadar — the internship intelligence platform
An automated, free-to-run web platform that scrapes internship listings across 20+ job platforms every X days, extracts in-demand skills using NLP, visualizes demand trends over time, and gives each student a personalized skill gap report — all hosted for ₹0/month.
1
Automated multi-platform scraping
GitHub Actions triggers Python scripts on a cron schedule. Indeed Publisher API, Internshala RSS, Remotive JSON, and Apify actors for Naukri/Shine feed fresh listings every X days automatically — no manual effort.
2
NLP-powered skill extraction
Every job description is processed by spaCy + a curated 500-skill keyword list. Skills are tagged, counted, and frequency-ranked across domains: AI/ML, Full Stack, Data Science, LLM/GenAI, Cybersecurity.
3
Time-series storage for trend data
Each scrape writes a dated snapshot to Supabase. This is what makes trend graphs possible — skill demand curves over weeks, months, and semesters. No competing free tool provides this.
4
Personalized skill gap analysis
A user enters their current skills. The platform scores every job against their profile, shows their match % per domain, and surfaces exactly which skills to learn next for the highest ROI.
5
Smart alerts — not noise
Instead of emailing every new job, Supabase Edge Functions only fire when a new listing matches a user's skill profile above a configurable threshold. Email via Resend or Telegram bot.
